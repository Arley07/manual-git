⬇️ Descargar e Instalar Git

Aprende cómo descargar e instalar Git según tu sistema operativo.

1️⃣ Descargar Git

Visita la página oficial de Git:

https://git-scm.com/downloads
  • Windows: Descarga el instalador “Git for Windows”.
  • macOS: Puedes instalarlo desde el sitio oficial o usando brew install git.
  • Linux: Usa sudo apt install git o sudo dnf install git según tu distribución.

2️⃣ Verificar instalación

Abre la terminal o PowerShell y ejecuta:

git --version

Si ves algo como git version 2.x.x, ¡ya está instalado correctamente!
